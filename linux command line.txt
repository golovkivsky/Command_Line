1. Create a directory named 'folder'

mkdir folder
cd folder

2. Creating 5 files test_A, Test_B ...

touch test_{A..E}

3. Recording the current date in each file
3.1.

date | tee test_{A..E}

3.2.

for t in test_{A..E}
do
date > $t
done

4. Count the number of lines '2017' in all files

grep '2017' test_{A..E} | wc -l

5. Remove all files

cd ..
rm -rf folder
